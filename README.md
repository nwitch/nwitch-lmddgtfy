# nwitch-lmddgtfy

nwitch plugin for [lmddgtfy](https://lmddgtfy.net/).

## Installation

``` bash
$ npm install --save nwitch-lmddgtfy
```

## Usage

``` toml
[plugins]
nwitch-lmddgtfy = true
```

``` irc
05:34 <KenanY> !lmddgtfy how do i javascript
05:34 <nwitch> KenanY: https://lmddgtfy.net/?q=how+do+i+javascript
```
